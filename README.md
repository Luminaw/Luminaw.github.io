# Luminaw.xyz

This is a my own personal page with docs, projects, and blogs. viewable at [Luminaw.github.io](https://luminaw.github.io/).

## Tech Stack

- Astro
- TailwindCSS
- Shadcn UI
- Pagefind
- MDX

## Features

- Landing Page
- Projects
- Blogs
- Documentation
- Search
- Tag Filtering
- Dark Mode
- Syntax Highlighting
- Responsive Design
