This is the docs on this blog.

### [This](https://github.com/Luminaw/Luminaw.github.io)

This is hosted on GitHub Pages (for now), and the source is available [here](https://github.com/Luminaw/Luminaw.github.io).

#### Core Tech Stack

This blog uses [Astro](https://astro.build/), [Tailwind CSS](https://tailwindcss.com/), and [MDX](https://mdxjs.com/).

#### Integrations

We use [Pagefind](https://pagefind.io/) for search and [Sitemap](https://astro.build/integrations/astro-sitemap/) for sitemap.

Along with that, we use rehypePrettyCode for syntax highlighting to look pretty.

And we adhear to GFM (GitHub Flavored Markdown) for markdown support.
